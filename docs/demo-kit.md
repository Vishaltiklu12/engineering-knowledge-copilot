# Recruiter Demo Kit

This folder contains the assets needed to present **Engineering Knowledge Copilot**
to recruiters, hiring managers, and technical reviewers as a real project instead
of just a code repository.

## What To Share

For the strongest public showcase, share these together:

1. **Live demo**
   - Public Swagger/API docs or a thin frontend over the query flow.
   - Seeded with sample documents so reviewers can test it immediately.
2. **30 to 60 second demo video**
   - Short walkthrough of upload, ingestion, query, and citation-backed answers.
3. **Architecture image**
   - Visual summary of API, worker, storage, and retrieval flow.
4. **Sample questions**
   - Curated prompts that quickly demonstrate grounding and retrieval quality.

## Recommended Public Demo Flow

1. Open the live demo URL.
2. Explain in one sentence what the project does.
3. Ask one guided sample question from [sample-questions.md](./sample-questions.md).
4. Show grounded answer output with citations.
5. Link to the repository and architecture diagram.
